Full-Stack Architecture Plan
HOW TO BUILD THIS FOR REAL
Complete technical roadmap to turn this prototype into a production-grade automated raffle system.
🏗️
Backend API
Node.js + Express REST API with PostgreSQL. Core tables: draws, entries, payments, winners. Use Redis for real-time entry count + WebSocket pub/sub. Each confirmed payment triggers an AUTO_ASSIGN function that returns a padded ticket number instantly.
Node.jsPostgreSQLRedisSocket.io
💳
PayMongo Integration
Create a PaymentIntent for ₱20 (or bulk price) on button click → redirect to PayMongo checkout. Set up a webhook endpoint at /webhooks/paymongo. When payment.paid fires, auto-confirm entry, assign ticket, send SMS/email. No manual GCash verification at all.
PayMongo API
📡
Real-Time Progress
Socket.io broadcasts ENTRY_UPDATE events to all connected clients after every payment webhook. The frontend's progress ring updates live — every participant sees the pool filling in real time. When entries hit target, server emits DRAW_TRIGGERED and the countdown auto-starts for all clients simultaneously.
WebSocketsRedis Pub/Sub
🎰
Draw & Roulette Engine
Server uses crypto.randomInt() (cryptographically secure) to pick the winner from confirmed entries. The result is sealed + timestamped before the roulette visually spins — ensuring no tampering. Roulette digits are calculated server-side and broadcast to all clients via WebSocket so everyone sees the same spin simultaneously.
📣
Notifications
Use Twilio SMS for: entry confirmation (ticket #), winner announcement, prize release confirmation. Optional: Facebook Messenger via facebook-messenger-api for in-app notifications to participants who joined via FB ad link. Auto-post winner to your Facebook page via Meta Graph API.
Meta Graph API
🖥️
Frontend Stack
React + Vite SPA. This prototype becomes your component library. Deploy on Vercel (free tier). Custom domain: dailydraw.ph. Facebook ad links point directly here. Mobile-first — 80%+ of your traffic will be from phones clicking FB ads. PWA manifest for "Add to Home Screen" stickiness.
